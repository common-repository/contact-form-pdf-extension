This plugin work with the contact form 7

== Description ==
This Plugin give you advantage to send the contact form data as attachment in PDF format.
You can make your email template to customize the data which is sending by the contact form 7.

== Installation ==
1. Upload Contact form 7 pdf extension to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= This plugin not working with file uploading? =
Right now plugin is overriding the default function of contact form 7 attachement that's why it not holding the file uploaded data. we will resolve it in feature.
